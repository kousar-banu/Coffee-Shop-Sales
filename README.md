# Coffee-Shop-Sales
This Excel project analyzes coffee shop sales data to understand overall sales performance. It highlights sales trends, top-selling products, and store-wise performance using pivot tables. Charts and dashboards are used for clear and easy data visualization.

#OBJECTIVE

The main objective of this project is to analyze retail sales data to gain actionable insights that will enhance the performance of the Coffee Shop.

# Recommeneded Analysis
.Analyze sales trends by day of the week and hour of the day to identify peak sales periods.
.Evaluate monthly total sales revenue to track growth and seasonal patterns.
.Compare sales performance across different store locations.
.Calculate the average order value per customer.
.Identify best-selling products based on quantity sold and total revenue.
.Analyze sales distribution by product category and product type?





# KPIs Included
 .Total Sales
 .Total Orders
 .Average Order Value
 .Top-Selling Products
 .Sales by Store
